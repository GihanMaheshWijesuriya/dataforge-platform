# Personalized Data Platform

A microservice-based backend platform for data ingestion, ETL, search, and personalized recommendations.

## Tech Stack
- Go — ingestion services
- Python — streaming & batch ETL
- Java Spring Boot — search & recommendation APIs
- Kafka — event streaming
- PostgreSQL — analytics store
- Elasticsearch — search
- DynamoDB — feature store
- Airflow — orchestration
- Docker — local infrastructure
